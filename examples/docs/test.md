## Icon 图标

提供了一套常用的图标集合。

### 使用方法

直接通过设置类名为 `z-icon-iconName` 来使用即可。例如：

:::demo
  ```html
  <template>
    <div>hello, i am md, {{count}}</div>
  </template>
  <script lang="ts">
  import { ref, defineComponent } from 'vue'
  export default defineComponent({
      name: 'test',
      data: () => {
          const count = ref(234);
          return { count };
      }
  })
  </script>
  <style>

  </style>
  <style>
    
  </style>
  ```
:::
